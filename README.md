# TFM-Bruno-Addiego
Recull de codi, funcions i sub-funcions pel predictor de paraules per pacients neurodegeneratius implementat en Raspberry Pi. Les llibreries requerides per a la seva execució són:

- TensorFlow
- NumPy
- Librosa
- Pandas
- PyAudio
- Flask

L'ús del sistema requereix d'un micròfon extern connectat per a la gravació d'àudio.
